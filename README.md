# 智慧警务调度系统

一个功能完善的警务调度指挥网站，用于实时监控警情、调度警力、管理任务和可视化警力分布。

## 🚔 功能特点

### 核心功能
- **实时警情监控** - 显示和处理各类警情
- **警力资源管理** - 警员状态、位置、车辆信息
- **地图可视化** - 实时显示警力分布和警情地点
- **任务调度系统** - 任务创建、分配和状态跟踪
- **数据统计报表** - 实时统计和报告生成

### 交互功能
- 紧急调度快速响应
- 警员状态实时更新
- 地图标记和导航
- 任务管理和编辑
- 实时通知系统

## 📁 文件结构

```
警务调度系统/

├── script.js           # 交互脚本
├── README.md           # 说明文档

```

## 🛠️ 技术栈

- **HTML5** - 语义化标签，现代结构
- **CSS3** - Flexbox、Grid布局，响应式设计
- **JavaScript (ES6)** - 纯前端交互
- **Leaflet.js** - 地图可视化库
- **Font Awesome** - 图标库
- **Google Fonts** - 中文字体优化

## 🚀 快速开始

2. 用浏览器打开 `index.html`
3. 无需任何服务器或安装

### 在线部署
推荐使用以下免费平台：
- **GitHub Pages** - 完全免费，适合静态网站
- **Vercel** - 一键部署，自动HTTPS
- **Netlify** - 拖拽上传，全球CDN
